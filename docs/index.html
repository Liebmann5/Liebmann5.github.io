Green Tea Press

GreenTeaPress is a website that offers free CS educational books by Allen B. Downey. These books are tremendous because of the vocabulary used throughout it! Every topic/subject is explained so easily that anyone who picks the book up will be able to easily comprehend the concept.

Example: (pg. 20)

Also, in mathematics, a statement of equality is true for all time. If a = b now, a is always equal to b. In Java, an assignment statement can make two variables equal, but they don't have to stay that way. 
          
            int a = 5;
            int b = a; // a and b are now equal
            a = 3; // a and b are no longer equal
The third line changes the value of a, but it does not change the value of b, so they are no longer equal. GreenTeaPress

Introduction to Programming

Introduction to Java is the next level up from GreenTeaPress's Intro book. In this one you go much more in-depth about everything working. It could be seen as an Intro book to everything involved in Computer Science. One should really take their time reading this book as everything in this book will be seen later down the road. Obtaining a strong understanding of the fundamentals early will seperate you from others also learning CS!
